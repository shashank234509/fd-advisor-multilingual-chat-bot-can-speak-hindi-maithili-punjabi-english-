Avatar Web UI Kaise Chalayein? (How to Run)

first set up the .env file and download the model from ollamma and put your api key in .env file 

Yeh naya 3D system FastAPI frontend aur browser ke microphone ka upyog karta hai. Ise chalana bahut aasaan hai.

Steps:

Terminal/Command Prompt kholiye aur apne is project ke folder (/Volumes/Download/chatbot 2) mein jayein.

Niche diya gaya command type karke enter dabayein:

python ui.py
(Aap is command ko directly terminal me run karenge to backend server start ho jayega. Terminal background me 'Uvicorn running on...' dikhayega).

Aapka jab server chalu ho jaye, toh apna Google Chrome browser kholiye.

Note: Hamesha Chrome browser use karein kyunki usme voice aur speech-to-text sabse achha aur seamlessly chalta hai.
URL search bar me exact yeh link type karein aur zaroor yahi link type karein: 👉 http://localhost:8000

❌ DHYAN RAHE: http://0.0.0.0:8000 ya file path na kholein, warna Chrome microphone chalne ki permission block kar dega. Hamesha localhost likhein.

Page open hote hi browser aapse Microphone use karne ki permission mangega. Usme "Allow" ka button dabayein. (Agar permission ka popup na aaye toh URL bar ke right side mein camera/mic icon par click karein aur permission dein).

Uske baad screen par "Start" button click karein aur aapki Avatar se baatcheet shuru ho jayegi!
